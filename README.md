## ğŸ”§ Customization

### Adding New Tools:

1. Create directory: `/{category}/{tool-slug}/`
2. Copy `tool_template.html` to `index.html`
3. Replace placeholders:
   - {{TITLE}} - Tool name
   - {{DESCRIPTION}} - Tool description
   - {{CATEGORY}} - Category name
   - {{SLUG}} - Tool slug
   - {{KEYWORDS}} - SEO keywords
4. Add tool-specific interface and JavaScript
5. Update category index page with new tool link
